Detailed Description

internal class to load a skinned model from Blender FBX

Note
to export skinned models from Blender properly: export as FBX, select all meshes and bone collection and apply "selected only", leave all armature settings as ticked
when adding normal maps be sure to use node>vector>normal map and feed the output into normal on the principle bsdf node
COLLADA files have the most reliable support at the moment
